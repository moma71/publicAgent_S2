🎓 AI 튜토리얼: Custom GPT로 전용 AI 비서 만들기

💡 Logic (원리)

Custom GPT는 특정 목적에 맞춘 전용 AI를 만드는 기능입니다. ChatGPT를 내 조직/업무에 맞게 커스터마이징하는 것이죠.

핵심 원리:

1. 역할 고정: "너는 대진시 민원 담당자야" 매번 안 써도 됨

2. 지식 주입: 조직 문서 업로드 (자동 RAG)

3. 프로세스 고정: 질문 → 검색 → 검증 → 답변 (자동)

기존 방식 vs Custom GPT:

기존: 매번 프롬프트 복붙 + 문서 업로드 Custom GPT: 한 번 설정 → 클릭만으로 사용

실전 효과:

- 윤서연의 프롬프트 100개 → Custom GPT 1개

- 매번 10분 → 3초

- 실수 가능성 감소

🛠️ Stack (도구)

필수:

- ChatGPT Plus (유료) - 월 $20

- 조직 문서 (PDF, TXT 등)

선택:

- Actions (API 연동, 고급)

📋 Guide (실행 단계)

Step 1: Custom GPT 생성

1. ChatGPT Plus 로그인 2. 좌측 메뉴 "Explore" 클릭 3. "Create" 버튼 클릭 4. "Configure" 탭 선택

Step 2: 기본 정보 입력

Name: 대진시 검증봇 Description: 대진시 민원 답변 + 자체 검증 Instructions (지시사항): " 너는 대진시 민원봉사과 AI 비서야. 역할: 1. 질문 받기 2. 업로드된 조례 문서 검색 (RAG) 3. 답변 생성 4. 자체 검증 (중요!) 5. 판단 근거 제시 답변 형식: === 답변 === [구체적 답변] === 판단 근거 === - 참고 조례: [조항] - 관련 법령: [법령] === 검증 결과 === ✓ 조례 일치 여부 ✓ 서류 완전성 ✓ 최신 규정 반영 처리 예상 시간: [N일] 담당 부서: [부서명] "

Step 3: 지식 업로드

Knowledge (지식): 1. "Upload files" 클릭 2. 조례 PDF 업로드 - 대진시_민원_조례_2024.pdf - 대진시_처리_사례집.pdf 3. 최대 10개 파일 (각 25MB)

Step 4: 대화 시작

Conversation starters (시작 질문 예시): 1. "전입신고 필요 서류는?" 2. "주차 위반 이의신청 절차는?" 3. "외국인 민원 처리 방법은?" 4. "민원 처리 기한은?"

Step 5: 테스트

"외국인 배우자 출생신고 필요 서류는?" → Custom GPT가 자동으로: 1. 조례 검색 2. 답변 생성 3. 검증 4. 근거 제시
