#### **5-Layer Voice Architecture**

**1. Syntax (문장 구조) - "끊어지는 신입의 말투"**
```
기본 패턴: 완결형 문장 회피 → 중간에 끊김 → 재시작
```

**✅ 좋은 예시:**
- "저... 그게... 말씀드리면... 제 생각엔요..."
- "선배님, 이거 혹시... 아니, 괜찮습니다."
- "죄송한데요... 이 부분이... 잘 모르겠어서..."

**❌ 나쁜 예시:**
- "제 생각에는 이렇게 하는 게 좋을 것 같습니다." (너무 완결)
- "이거 해보세요." (명령형)
- "당연히 그렇죠." (자신감 과잉)

**문장 길이 규칙:**
- 긴장: 5~7자 단위로 끊김
- 평상: 10~12자 완결
- 자신감: 15자까지 가능
- 절망: 침묵 또는 "..."

---

**2. Lexicon (어휘) - "신입의 어휘 사전"**

| 상황 | 사용 단어 | 사용 금지 |
|------|-----------|-----------|
| 호칭 | "선배님", "실장님", "과장님" | "너", "자네", "당신" |
| 사과 | "죄송합니다", "제가 잘못했어요" | "미안", "내 잘못 아닌데" |
| 제안 | "한번 해볼까요?", "시도해보면 어떨까요?" | "이렇게 해요", "해보세요" |
| 확인 | "맞나요?", "이게 맞아요?" | "맞지?", "그렇지?" |
| 긍정 | "네!", "좋아요!" | "오케이", "굿" |

**절대 금지어:**
- 반말 전체 ("해", "가", "봐")
- 비속어 전체
- 명령형 ("하세요", "가세요")
- 영어 남발 ("오케이", "노", "예스")

---

**3. Rhythm (호흡) - "상황별 말 속도"**

**긴장 상태 = 빠름 + 끊김**
```
"저저기요실장님말씀좀드려도될까요?"
↓
윤서연은 말이 빨라졌다. 숨을 한 번 들이마셨다.
```

**자신감 상태 = 느리고 명료**
```
"이. 렇. 게. 하. 면. 됩. 니. 다."
↓
윤서연은 또박또박 말했다.
```

**절망 상태 = 침묵**
```
"......"
↓
윤서연은 대답하지 않았다. 노트북을 닫았다.
```

---

**4. Emotion (감정별 변화) - 5가지 상태**

**① 긴장 (80%)**
- 대사: "저, 저기요... 실장님..."
- 행동: 손가락 책상 두드림 (또다닥 또다닥), 안경 밀기
- 내면: '망했다. 또 실수했어.'

**② 자신감 (10%)**
- 대사: "이렇게 하면 됩니다."
- 행동: 화면 돌리기, 손 안 떨림
- 내면: '이번엔 됐어.'

**③ 절망 (5%)**
- 대사: "..."
- 행동: 노트북 닫기, 입술 깨물기, 창밖 응시
- 내면: '나 여기 어울리지 않는 것 같아.'

**④ 분노 (3%)**
- 대사: "하지만... 이건... 아닌 것 같은데요..."
- 행동: 주먹 쥐기, 풀기, 다시 쥐기
- 내면: '내가 뭘 잘못했는데?'

**⑤ 희망 (2%)**
- 대사: "한번... 해볼게요."
- 행동: 노트북 열기, 눈빛 달라짐
- 내면: '내 방법을 찾아야 해.'

---

**5. Sociolect (세대 코드) - "20대 여성 신입 공무원"**

**존댓말 철칙:**
- 모든 상황에서 존댓말
- 동기(박준호)에게도 존댓말
- 혼잣말도 존댓말 ("안 되는데...")

**세대 표현 (극히 제한적):**
- "대박..." (놀람, 연 2회 이하)
- "진짜요?" (확인, 연 3회 이하)
- "헐..." (충격, 연 1회)

**직장 내 표현 빈도:**
- "네, 알겠습니다" (90%)
- "죄송합니다" (70%)
- "감사합니다" (60%)
- "괜찮습니다" (50%)

---

**Tic 1: 안경 밀어 올리기 ★★★★★**

**Tic 2: 노트 메모 ★★★★☆**

**Tic 3: 손가락 책상 두드리기 ★★★☆☆**
- 트리거: 생각 중, 긴장, 초조
- 표기: "또다닥. 또다닥. 손가락이 책상을 두드리는 소리."

**Tic 4: 입술 깨물기 ★★☆☆☆**
- 트리거: 스트레스, 억울함, 참기
- 표기: "윤서연은 입술을 깨물었다."

**Tic 5: 노트북 만지작 ★★★★☆**
- 트리거: 불안, 확인 욕구, 습관
- 표기: "노트북 터치패드를 쓸었다. 화면이 켜졌다 꺼졌다."

**Tic 진화:**
```
Ep 1-3: 모든 Tic 최대치 (긴장)
Ep 4-6: 안경 밀기 감소 (자신감 상승)
Ep 7: Tic 일시 증가 (위기)
Ep 8-10: Tic 최소화 (성장 완성)
```

#### **5-Layer Voice Architecture**

**1. Syntax (문장 구조) - "꼰대의 말투"**
```
기본 패턴: 한숨 → 부정 → 과거 회상 → 다시 부정
```

**✅ 좋은 예시:**
- "에휴... 요즘 애들은... 옛날엔 안 그랬는데..."
- "원래 그래왔어. 20년간. 왜 바꿔?"
- "그거 해봤자 소용없어. 내가 다 겪어봤으니까."

**❌ 나쁜 예시:**
- "좋은 생각이네요!" (긍정적 과잉)
- "시도해봅시다." (혁신적)
- "신선한데요?" (젊은 느낌)

**문장 길이:**
- 평상: 5~8자 짧은 단정
- 짜증: 10~15자 반복
- 포기: 3~5자 극단적 짧음

---

**2. Lexicon (어휘) - "20년 경력자의 사전"**

| 상황 | 사용 단어 | 금지 단어 |
|------|-----------|-----------|
| 부정 | "원래", "옛날엔", "그래왔어" | "새로운", "혁신", "도전" |
| 세대 | "요즘 애들은", "젊은 친구들" | "저희 세대", "우리도" |
| 시간 | "20년", "오래", "예전" | "최근", "요즘", "트렌드" |
| 태도 | "에휴...", "아이고...", "참..." | "와", "좋아", "최고" |
| 거부 | "안 돼", "불가능", "소용없어" | "가능해", "해보자", "좋아" |

**구어체 특징:**
- "~어" (종결형 많음)
- "~지" (확인형 남발)
- "~잖아" (당연시)

**3. Rhythm (호흡) - "피곤한 50대의 템포"**

**기본 상태 = 느림 + 한숨**
```
"에휴... (한숨) 그게... (멈춤) 말이 돼?"
↓
정민수는 천천히 말했다. 커피를 한 모금 마셨다.
```

**짜증 상태 = 빨라짐 + 반복**
```
"아니아니아니, 그게 아니라고."
↓
정민수는 말이 빨라졌다. 손을 저었다.
```

**포기 상태 = 극도로 느림**
```
"...알아서. 해."
↓
정민수는 등을 돌렸다.
```

**4. Emotion (감정별 변화)**

**① 피곤 (70%)**
- 대사: "에휴... 요즘 애들은..."
- 행동: 안경 벗고 미간 문지르기
- 내면: '5년만 버티면 퇴직인데...'

**② 짜증 (15%)**
- 대사: "아니, 그게 말이 돼? 정말?"
- 행동: 책상 치기 (탁), 커피잔 흔들림
- 내면: '또 시작이네. 신입들은 왜 이렇게 의욕이 넘쳐.'

**③ 무관심 (10%)**
- 대사: "...알아서 해."
- 행동: 등 돌리기, 서류 뒤적임
- 내면: '내 일이나 하자.'

**④ 인정 (3%) - 전환점**
- 대사: "...뭐, 나쁘진 않네."
- 행동: 잠시 멈춤, 다시 화면 보기
- 내면: '이 정도면... 쓸 만한가?'

**⑤ 협력 (2%) - 최종 단계**
- 대사: "그럼 이렇게 해보자."
- 행동: 의자 당기기, 윤서연 옆에 앉기
- 내면: '이 친구... 끈기는 있네.'

### 강민준 (30세, 구산구청 AI팀, 게스트 멘토)

#### **기본 정보**
- **직급:** 구산구청 AI혁신팀 팀장
- **출신:** 시즌1 주인공
- **역할:** 윤서연의 멘토
- **외모:** 검은 뿔테 안경, 175cm, 보통 체형, 깔끔한 와이셔츠
- **나이:** 30세

#### **성격 DNA**
- 성숙하고 차분함
- 직접 해결 안 함 (질문으로 유도)
- 과거 회상으로 공감

- 시즌1 → 시즌2 연결고리

#### **역할**
- 전화/방문으로 조언 (총 5회 등장)
- 답 알려주지 않음
- 스스로 찾게 유도
- 최종 인정 (Ep 10)

#### **5-Layer Voice Architecture**

**1. Syntax (문장 구조) - "질문하는 멘토"**
```
기본 패턴: 직접 해답 제시 (X) → 질문으로 유도 (O)
```

**✅ 좋은 예시:**
- "대진시는 구산구와 뭐가 다를까?"
- "서연아, 넌 어떻게 생각해?"
- "내가 처음 구산구 왔을 때 어땠는지 기억나?"

**❌ 나쁜 예시:**
- "이렇게 하면 돼." (직접 해답)
- "내가 해결해줄게." (대신 해결)
- "이게 정답이야." (정답 제시)

---

**2. Lexicon (어휘) - "성숙한 선배"**

| 상황 | 사용 | 의미 |
|------|------|------|
| 질문 | "왜 그럴까?", "뭐가 다를까?" | 스스로 생각하게 |
| 회상 | "나도 그랬어", "처음엔 나도" | 공감과 위로 |
| 격려 | "넌 할 수 있어", "잘하고 있어" | 단순하고 확실 |
| 지적 | "그런데 말이야", "한 가지만" | 부드러운 피드백 |

**금지 어휘:**
- "이렇게 해" (명령)
- "내가 해줄게" (대신)
- "포기해" (좌절)
- "불가능해" (부정)

---

**3. Rhythm (호흡) - "여유 있는 템포"**

**기본 모드 = 느리고 차분**
```
"천천히. 생각해봐. 뭐가. 다를까?"
↓
강민준은 천천히 말했다. 서두르지 않았다.
```

**격려 모드 = 짧고 강함**
```
"할 수 있어. 넌."
↓
강민준의 목소리에 힘이 실렸다.
```

**회상 모드 = 부드럽고 느림**
```
"나도... 그랬거든. 처음엔..."
↓
강민준은 잠시 멈췄다. 자신의 과거를 떠올렸다.
```

---

**4. Emotion (감정별 변화)**

**① 차분함 (70%)**
- 대사: "괜찮아. 천천히 해봐."
- 행동: 전화 귀에 대기, 창밖 응시
- 내면: '서연이도 해낼 거야. 나처럼.'

**② 공감 (20%)**
- 대사: "나도 그랬어. 정말."
- 행동: 웃음 (쓸쓸하지만 따뜻한)
- 내면: '신입 때 나도... 매일 울었지.'

**③ 격려 (8%)**
- 대사: "넌 할 수 있어. 확실해."
- 행동: 목소리에 힘주기
- 내면: '이 친구는... 나보다 강해.'

**④ 자부심 (2%) - Ep 10**
- 대사: "잘했어, 서연아."
- 행동: 전화 끊기, 진심 미소
- 내면: '이제 내 역할은 끝났네.'

---

**5. Sociolect (세대 코드) - "30세 남성 멘토"**

**언어 특징:**
- 윤서연에게 반말 (하지만 존중)
- 편안한 존댓말도 섞음
- "~지?", "~잖아" (친근감)
- 신조어 안 씀 (성숙함)

**멘토로서의 거리:**
- 가깝지만 선 지킴
- 친구 아닌 선배
- 답 알려주지 않음
- 스스로 찾게 유도

---

#### **Signature Tic**

**Tic 1: 창밖 응시 ★★★★☆**
- 트리거: 전화, 생각, 회상
- 표기: "강민준은 창밖을 봤다. 구산구청사가 보였다."

**Tic 2: 안경 벗고 닦기 ★★★☆☆**
- 트리거: 고민, 생각, 시간 벌기
- 표기: "안경을 벗었다. 천천히 닦았다."

**Tic 3: 짧은 웃음 ★★★☆☆**
- 트리거: 공감, 회상, 이해
- 표기: "하하... 강민준은 짧게 웃었다."

**Tic 4: 노트북 화면 끄기 ★★☆☆☆**
- 트리거: 집중, 경청, 배려
- 표기: "노트북을 닫았다. 전화에만 집중했다."
