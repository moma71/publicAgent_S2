## 🎓 AI 튜토리얼: 검증 프롬프트로 오류 잡기

### 💡 Logic (원리)

**검증 프롬프트**는 AI의 답변을 AI가 다시 검증하는 기법입니다. 

**핵심 원리:**
1. **더블 체크**: AI가 답변 후 스스로 재검증
2. **Cross Check**: 다른 각도에서 질문
3. **Logic Check**: 모순 찾기

**왜 필요한가?**
```
AI도 실수함:
- 문서 누락 → 잘못된 답변
- 오래된 정보 → 과거 답변
- 애매한 질문 → 부정확한 해석
```

**실전 효과:**
- 윤서연의 오류 1건 → 검증 후 0건
- 99.976% → 100% 목표
- 신뢰도 상승

### 🛠️ Stack (도구)

**필수:**
- ChatGPT (무료/유료)
- 검증용 프롬프트 템플릿

**선택:**
- Claude (교차 검증용)

### 📋 Guide (실행 단계)

**Step 1: 기본 답변 생성**

```
프롬프트:
"대진시 전입신고 필요 서류는?"

AI 답변:
"주민등록등본, 신분증, 전입신고서"
```

**Step 2: 검증 프롬프트 실행**

```
검증 프롬프트:
"위 답변을 검증해줘.

검증 항목:
1. 대진시 조례 제7조와 일치하는가?
2. 필요 서류가 빠진 게 없는가?
3. 최신 규정(2024년)인가?

검증 결과 형식:
- 정확도: O/X
- 누락 항목: [있으면 명시]
- 수정 필요: [있으면 제안]"
```

**Step 3: 결과 확인**

```
검증 결과:
- 정확도: X
- 누락 항목: 가족관계증명서
- 수정 필요: "가족관계증명서 추가 필요"

최종 답변:
"주민등록등본, 신분증, 전입신고서, 가족관계증명서"
```

### 🎯 Prompt (바로 쓰는 템플릿)

**기본 검증 템플릿:**

```
[1단계: 질문]
질문: [여기에 질문 입력]

[2단계: AI 답변 받기]
(AI가 답변)

[3단계: 검증 프롬프트]
위 답변을 다음 기준으로 검증해줘:

검증 기준:
1. 사실 정확성: 틀린 정보가 있는가?
2. 완전성: 빠진 정보가 있는가?
3. 최신성: 과거 정보는 아닌가?
4. 일관성: 모순되는 내용은 없는가?

검증 결과:
- 정확도: [O/X]
- 문제점: [있으면 명시]
- 수정안: [있으면 제안]
```

- 너무 많으면 비효율
- 중요한 결정은 반드시 검증
- 일상적 질문은 1단계만
