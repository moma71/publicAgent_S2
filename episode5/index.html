🎓 AI 튜토리얼: Canvas로 실시간 협업하기

💡 Logic (원리)

Canvas는 ChatGPT의 협업 문서 편집 기능입니다. 여러 명이 동시에 문서를 보고, 수정하고, AI가 실시간으로 정리합니다.

핵심 원리:

1. 실시간 편집: 여러 명이 동시에 문서 수정

2. AI 정리: ChatGPT가 내용을 자동 요약/정리

3. 버전 관리: 수정 이력 자동 저장

기존 방식 vs Canvas:

기존: 회의록 수기 작성 → 타이핑 → 공유 → 수정 요청 → 재작성 Canvas: AI 자동 정리 → 실시간 수정 → 즉시 공유 → 끝

실전 효과:

- 회의록 작성 시간 70% 단축

- 오타/누락 자동 감지

- 참석자 전원 동시 확인

🛠️ Stack (도구)

필수:

- ChatGPT Plus (유료) - 월 $20 (약 2만 원)

- 웹 브라우저만 있으면 OK

선택:

- 구글 독스 연동 가능

- Notion 내보내기 가능

📋 Guide (실행 단계)

Step 1: Canvas 열기

1. ChatGPT Plus 로그인 2. 새 대화 시작 3. 프롬프트 입력창 옆 "Canvas" 아이콘 클릭 (또는 프롬프트에 "Canvas로 만들어줘" 입력)

Step 2: 문서 생성

프롬프트 예시: "회의록 양식을 Canvas로 만들어줘. 포함 항목: - 회의 일시/장소 - 참석자 - 안건 - 주요 의견 - 결정사항 - 다음 액션 아이템"

→ AI가 자동으로 표 형식 문서 생성

Step 3: 실시간 편집

회의 중: 1. Canvas 화면 공유 (Zoom/Teams) 2. 참석자 발언 → AI에게 입력 3. AI가 자동 정리 → 실시간 반영 4. 참석자가 즉시 확인/수정

Step 4: 자동 정리 요청

프롬프트: "위 회의 내용을 다음 형식으로 정리해줘: 1. 핵심 결정사항 3가지 2. 각 참석자별 주요 의견 3. 다음 주까지 해야 할 액션 아이템"

→ AI가 3초 안에 구조화된 정리본 생성

🎯 Prompt (바로 쓰는 템플릿)

기본 회의록 템플릿:

Canvas로 회의록을 만들어줘. [회의 정보] - 일시: 2025년 12월 10일 14:00 - 장소: 대진시청 3층 회의실 - 참석자: 정민수, 윤서연, 이현주, 박준호 [안건] 1. AI 교육 프로그램 기획 2. 예산 확보 방안 3. 교육 일정 수립 실시간으로 내용을 입력할 테니, 자동으로 정리해줘. 각 발언자별로 구분하고, 핵심만 추출해줘.

실전 활용 예시 1 (프로젝트 회의):

Canvas로 프로젝트 킥오프 회의록 만들어줘. [프로젝트명] AI 민원 시스템 구축 [기간] 2025.12.01 ~ 2026.02.28 포함 내용: - 프로젝트 목표 (SMART 기준) - 주요 마일스톤 5개 - 팀원별 역할/책임 - 리스크 요인 - 다음 회의 일정 표 형식으로 만들고, 각 항목에 체크박스 넣어줘.

실전 활용 예시 2 (브레인스토밍):

Canvas로 아이디어 보드 만들어줘. [주제] 민원 처리 시간 단축 방안 형식: 1. 아이디어 목록 (번호, 제안자, 내용) 2. 각 아이디어의 장단점 3. 실현 가능성 (상/중/하) 4. 우선순위 투표 우리가 아이디어를 입력하면, 자동으로 분류하고 정리해줘.

실전 활용 예시 3 (보고서 초안):

Canvas로 시장 보고서 초안 작성해줘. [제목] 대진시 AI 민원 시스템 도입 성과 보고 [구조] 1. 요약 (3줄) 2. 배경 및 목적 3. 주요 성과 (숫자 중심) - 처리 시간 단축: 25% - 민원 만족도: 상승 - 직원 업무 부담: 감소 4. 향후 계획 5. 예산 소요 공문서 형식, 존댓말, A4 2페이지 분량으로.
