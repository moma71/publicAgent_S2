🎓 AI 튜토리얼: 데이터 분석으로 설득하기

💡 Logic (원리)

데이터 분석 = 숫자로 말하기입니다. 감정이 아닌 팩트로 증명하는 것이죠.

핵심 원리:

1. 시각화: 숫자 나열 < 그래프 1개

2. 비교: Before & After 명확히

3. ROI: 투자 대비 효과 계산

말 vs 데이터:

말: "AI 도입하니 좋아요" 데이터: "처리 시간 25% 단축, 연간 2천만 원 절감"

실전 효과:

- 윤서연의 시의회 통과

- 반대파 설득

- 예산 추가 확보

🛠️ Stack (도구)

필수:

- ChatGPT (무료/유료)

- 엑셀 파일 또는 CSV

선택:

- ChatGPT Advanced Data Analysis (유료)

- 구글 시트 연동

📋 Guide (실행 단계)

Step 1: 데이터 준비

엑셀 파일 예시: - 파일명: 민원통계_2025.xlsx - 시트: 월별 처리 시간, 민원 유형, 만족도 | 월 | 처리시간(분) | 처리건수 | 만족도 | |----|-------------|---------|--------| | 10월 | 27 | 1,200 | 72 | | 11월 | 22 | 1,350 | 81 | | 12월 | 20 | 1,400 | 89 |

Step 2: ChatGPT에 업로드

1. ChatGPT 열기 2. 클립 아이콘으로 엑셀 업로드 3. "이 데이터를 분석해줘" 명령

Step 3: 분석 요청

프롬프트: "이 데이터로 다음 분석해줘: 1. 월별 처리 시간 변화율 2. 처리 건수 증가율 3. 만족도 상승률 4. 투자 대비 효과 (ROI) 결과를 다음 형식으로: - 핵심 수치 3가지 - 그래프 3개 (막대, 선, 원) - 1줄 요약"

Step 4: 그래프 생성

프롬프트: "위 데이터를 시각화해줘. 그래프 1: 월별 처리 시간 (막대 그래프) - X축: 월 - Y축: 시간(분) - 색상: 파란색 그래프 2: 만족도 추이 (선 그래프) - X축: 월 - Y축: 만족도(점) - 목표선: 85점 (빨간 점선) 그래프 3: 투자 효과 (원 그래프) - 절감 효과: 50% - 추가 가치: 30% - 기타: 20%"

→ AI가 3초 안에 그래프 생성

🎯 Prompt (바로 쓰는 템플릿)

기본 데이터 분석 템플릿:

[엑셀 파일 업로드: 민원통계_2025.xlsx] 이 데이터를 분석해서 보고서 초안을 작성해줘. [분석 목표] AI 시스템 도입 효과 검증 [분석 항목] 1. 처리 시간 단축 효과 2. 처리 건수 증가율 3. 시민 만족도 변화 4. 투자 대비 효과 (ROI) [출력 형식] - 요약 (3줄) - 핵심 수치 5개 - 그래프 3개 - 결론 (투자 가치) [그래프 요구사항] - 막대 그래프: 월별 비교 - 선 그래프: 추세 분석 - 원 그래프: 구성 비율 보고서는 A4 2페이지, 공문서 형식으로.

💡 설득력 높이는 법

Rule 1: 비율보다 절대값

❌ "25% 단축" ✅ "25% 단축 = 연간 140시간 = 직원 0.5명분"

Rule 2: 비교 대상 명확히

Rule 3: ROI 계산 필수
