🎓 AI 튜토리얼: 실시간 통합 시연

💡 Logic (원리)

실시간 통합 시연은 여러 AI 기술을 실전 상황에서 동시에 활용하는 것입니다.

핵심 원리:

1. Multi-Tool: 여러 도구 동시 사용

2. Real-time: 즉각적 대응

3. Backup: 오류 시 대체 방안

왜 필요한가?

[상황] 예상치 못한 질문 [문제] 한 가지 방법으로 안 됨 [해결] 여러 방법 조합

🛠️ Stack (도구)

필수 도구:

1. ChatGPT (기본 대화)

2. Custom GPT (전용 봇)

3. Canvas (실시간 편집)

4. 검증 프롬프트 (더블 체크)

조합 예시:

1단계: Custom GPT로 1차 답변 ↓ 2단계: 검증 프롬프트로 체크 ↓ 3단계: Canvas로 실시간 수정 ↓ 4단계: 최종 답변 제출

📋 Guide (실행)

**Step 1: 준비 단계**

1-1. 도구 체크리스트

□ ChatGPT Plus 로그인 □ Custom GPT 테스트 □ Canvas 열어두기 □ 검증 프롬프트 저장

1-2. 시나리오 준비

예상 질문 10개 작성 → 각 질문별 답변 미리 테스트 → 오류 발생 시 대체 방안

**Step 2: 실전 시연**

2-1. 1차 시도 (Custom GPT)

[입력] "외국인 출생신고 베트남어" [출력] (Custom GPT 자동 답변) [체크] 답변이 맞나? → 검증 단계로

2-2. 검증 단계

[검증 프롬프트] 위 답변을 다시 한번 검토해줘. 1. 조례 조항이 맞나? 2. 필요 서류가 완전한가? 3. 절차가 정확한가? [결과] ✓ 모두 정확 → 제출 ✗ 오류 발견 → 수정 단계로

2-3. 수정 단계 (Canvas)

Canvas 열기 → 답변 붙여넣기 → 오류 부분 실시간 수정 → 팀원과 동시 편집 → 최종본 완성

**Step 3: 긴급 상황 대응**

3-1. 도구 오류 시

Custom GPT 안 됨 ↓ 일반 ChatGPT로 전환 ↓ 수동 프롬프트 입력

3-2. 시간 부족 시

검증 단계 생략 ↓ 1차 답변 바로 제출 ↓ 사후 보완 준비

3-3. 예상 밖 질문

당황하지 말기 ↓ 기본 템플릿 활용 ↓ "확인 후 답변 드리겠습니다"

💬 Prompt (템플릿)

**템플릿 1: 경진대회 시연용**

당신은 [우리 지자체] 민원 처리 AI입니다. [역할] - 민원인 질문에 정확히 답변 - 조례 조항 반드시 명시 - 처리 절차 단계별 안내 [제약] - 답변 시간: 3분 이내 - 언어: 한국어/영어/중국어/베트남어 - 형식: 조례 근거 + 서류 목록 + 절차 [예시] 질문: 전입신고 방법? 답변: === 필요 서류 === 1. 전입신고서 (조례 제3조) 2. 신분증 3. 가족관계증명서 (조례 제5조) === 처리 절차 === 1단계: 서류 준비 2단계: 민원실 방문 3단계: 제출 (처리 시간 10분) 이제 질문해주세요.

**템플릿 2: 실시간 검증용**

위 답변을 3단계로 검증해줘: [1단계] 더블 체크 - 조례 조항이 정확한가? - 필요 서류가 빠진 게 없나? [2단계] 크로스 체크 - 다른 조례와 충돌하지 않나? - 최신 법령이 맞나? [3단계] 로직 체크 - 절차 순서가 맞나? - 처리 시간이 현실적인가? 검증 결과를 아래 형식으로: ✓ [항목]: 정확 ✗ [항목]: 오류 발견 (이유)

**템플릿 3: 긴급 수정용**

아래 답변에서 [문제점]을 수정해줘: [원본 답변] (기존 답변 붙여넣기) [문제점] - 조례 제7조 누락 - 베트남어 번역 필요 [요구사항] - 조례 제7조 추가 - 전체 베트남어로 번역 - 형식 유지 즉시 수정본 제시.
